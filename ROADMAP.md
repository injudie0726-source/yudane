## 💰 マネタイズ戦略：忖度なしの評価

### 案1: 引き直し機能（課金）

**ユーザー提案**: 1日1回の制限を、課金で引き直せるようにする

#### 🟢 おすすめ度: ★★★★☆（4/5）

| 観点 | 評価 |
|------|------|
| **収益性** | ◎ ガチャ心理と相性抜群。SSR欲しさに課金する人は確実にいる |
| **実装難度** | ◎ 簡単。Stripe Checkout で30分で実装可能 |
| **UX影響** | △ やりすぎると「1日1回」の神聖さが薄れる |
| **法的リスク** | ◎ 景品表示法的にも問題なし（有料でレア確定ではない） |

#### 💡 おすすめ実装方法

```
【無料ユーザー】
└─ 1日1回無料

【引き直しオプション】
├─ 動画広告視聴 → 1回無料で引き直し（1日1回まで）
├─ ¥120 課金 → 即座に引き直し（1日3回まで）
└─ ¥500/月 サブスク → 無制限引き直し + 広告非表示
```

**注意**: 「無制限引き直し」は1日5回程度にキャップ推奨。
→ 本当の無制限だと「SSR出るまで引く」で価値が下がる

### 案2: 神託ワード連動アフィリエイト

**ユーザー提案**: 神託の内容に関連した広告をアフィリエイトで表示

#### 🟡 おすすめ度: ★★★☆☆（3/5）

| 観点 | 評価 |
|------|------|
| **収益性** | △ アフィリエイト単価は低い（1クリック数円〜数十円） |
| **実装難度** | △ キーワード→商品のマッピングが必要で手間 |
| **UX影響** | ◎ 世界観に溶け込めば「運命の導き」として受け入れられる |
| **相性** | ◎ コンセプト的には神託と広告の親和性は高い |

#### 🚫 正直な問題点

1. **日本のアフィリエイトは承認制が多い**
   - Amazonアソシエイト、楽天アフィリエイトは個人でも可
   - A8.net等は審査あり、サイト実績が必要

2. **キーワードマッチングの精度問題**
   - 「アイスを食べながら」→ アイスのアフィ？どの商品？
   - 動的マッチングはAIが必要でコスト増

3. **単価が低い**
   - Amazon: 購入額の2-8%
   - 1,000円の商品で20-80円程度

#### 💡 おすすめ代替案：「神託に導かれしアイテム」

アフィリエイトより**キュレーション型EC紹介**がおすすめ：

```
【神託表示後に表示】

━━━━━━━━━━━━━━━━━━━━━━━━
🌙 神託が導くアイテム
━━━━━━━━━━━━━━━━━━━━━━━━

今日の神託:「布団をかぶって、そのまま寝落ちしてください。」

→ あなたに必要なもの:
   🛏️ 無印良品 羽毛布団 [Amazonで見る]
   ☕ スタバ デカフェ [楽天で見る]
   🎧 睡眠導入ASMR [Spotify]

━━━━━━━━━━━━━━━━━━━━━━━━
```

**実装方法**:
1. 神託Part C（結論）ごとに「おすすめアイテム3つ」を事前定義
2. 24パターン × 3アイテム = 72リンクを手動で用意
3. Amazonアソシエイト + 楽天アフィリエイトで開始

**予想収益**: DAU 10,000 × CTR 3% × CVR 5% × 単価50円 = **¥7,500/月**
→ 正直、メイン収益にはならない。あくまで補助。
